# VDK ( Vite Development Kit )

### Installed Packages

#### Props Validate
```bash
npm i prop-types
```

#### Icons
```bash
npm i lucide-react
```

#### Routing
```bash
npm i react-router-dom
```

#### Fonts
```bash
npm i @fontsource/montserrat
npm i @fontsource/poppins
npm i @fontsource/roboto
```

#### Styling
```bash
# Tailwind CSS
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```
